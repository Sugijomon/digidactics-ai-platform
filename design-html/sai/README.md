# SAI Design HTML

High-fidelity HTML prototypes for the Shadow AI Scan.

These files are visual references, not production code. Production implementation belongs in `apps/sai`.

## Structure

```txt
survey/
  screen-01-intro.html
  screen-02-werkplek.html
  screen-03-frequentie.html
  screen-04-toolpicker-fixed.html
  screen-05-datatype.html
  screen-06-accountmatrix.html
  screen-07-vaardigheid-spelregels.html
  screen-08-toekomst.html
  screen-09-afronding.html
  DESIGN_SAI_COMPONENTEN.html

dpo-dashboard/
  reserved for Activatie, Tool Inventaris, Risicoprofiel, Governance, and Rapportage prototypes
```

## Product Boundary

The survey prototypes describe the respondent experience:

- simple and psychologically safe
- step-by-step
- mobile-friendly
- optimized for honest self-reporting
- writes structured scan data

The DPO dashboard prototypes describe the analysis experience:

- dense operational interface
- filters, matrices, drawers, and triage
- reads and interprets scan data
- supports DPO review and governance decisions
